## 🛠️ 技能專長詳細分析

### 💻 前端 JS 框架與工具
**🎯 核心理念：**
React 和 Vue 雙刀流，兩種工具皆有開發完整產品的經驗，同時只要沒有特別情況皆使用 TypeScript 撰寫。

API 除了 Restful 以外，也有在大型專案維護 GraphQL 的經驗。

**📦 技能清單：**
- **核心框架：** TypeScript, Vue, NuxtJS, Vue Router, Vuex, Vuelidate, VeeValidate, React, NextJS, MobX, React Hook Form, Yup
- **建構工具：** Webpack
- **圖表工具：** Chartjs
- **編輯器：** Quill
- **即時通訊：** GraphQL, Socket.io
- **HTTP 請求：** Axios
- **國際化：** i18n
- **傳統工具：** jQuery
- **測試工具：** Jest, Cypress.io
- **實用函式庫：** Lodash
- **雲端服務：** Firebase
- **效能工具：** Lighthouse
- **依賴管理：** Renovate
- **文件工具：** Storybook

**🎯 核心理念：**
主力使用 Sass + TailwindCSS + BEM 為專案打造專屬的小型、可複用的 UI 框架，如此架構可在專案設計上帶來更大的彈性，避免不斷使用同一框架造成的審美疲勞。

**⚡ 快速開發：**
只有當專案對設計要求不高時，習慣使用 Bootstrap、ElementUI、Buefy 快速切版。

**📦 技能清單：**
- **預處理器：** Sass, Scss
- **現代化工具：** TailwindCSS
- **傳統框架：** Bootstrap
- **Vue 生態：** ElementUI, Buefy, Vuetify
- **現代 CSS：** CSS variable, Media query
- **方法論：** BEM Naming convention

### 📱 PWA (Progressive Web App)
**🎯 技術理念：**

**📦 技術工具：**
- **資源庫：** Workbox、web-push
- **客戶端資料庫：** IndexedDB、Cache Storage
