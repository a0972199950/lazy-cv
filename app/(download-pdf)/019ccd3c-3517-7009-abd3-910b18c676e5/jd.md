# 【聯盟管理系統】前端工程師 (台中市西區)

## 公司：緯雲股份有限公司 (Cloud Latitude Software)

- **職位類型**：全職 / 初階
- **地點**：臺中市, 台灣
- **經驗要求**：需具備 1 年以上工作經驗
- **薪資**：50,000 ~ 100,000 TWD / 月
- **標籤**：台中, 前端, React.js
- **特色**：外籍人才、友善轉職人士

## 職缺描述

我們的聯盟管理系統是一個新開啟的專案。這是一個為了各運動聯盟補足 IT 能力的解決方案。此平台會包含聯盟賽程與球員表現紀錄、資料呈現，與 API 產品等。我們將採敏捷開發的精神，將團隊的工作時間專注在最重要的工作上，減少重工與浪費。歡迎您成為團隊一份子！

您的工作內容將包含：
- 設計與撰寫遊戲與後台網頁（含單元測試）
- 串接後端 API
- Code Review 同事的程式
- 自動化部署與維護

本職位的工作者需要常態性地：
- 與同事進行 Pair Programming
- 公開 Review 彼此的 Code
- 在有需要時撰寫後端的功能

## 職務需求

### 必要條件：
- 能從用戶痛點發始分析，提出 Solution 後進行架構設計
- 熟悉 VueJS、ReactJS、AngularJS 任一框架（工作時將以 React 為主）
- 了解版控工具，如 Git
- 有閱讀官方技術文件的能力

### 加分條件：
- 有單元測試經驗
- 使用過 Docker 與 CI/CD工具
- TypeScript、Tailwind 使用經驗

### 專業素養：
- 持續自主學習
- 可與團隊合作
- 善於解決問題
- 勇於改善流程
