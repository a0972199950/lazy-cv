## 🚀 部署說明

### 開發環境
```bash
# 安裝依賴
pnpm install

# 啟動開發伺服器
pnpm dev

# 訪問履歷
http://localhost:3000/[uuid]/[locale]
```

### 生產環境
```bash
# 建構生產版本
pnpm build

# 啟動生產伺服器
pnpm start
```
